# NYC Yellow Taxi Executive Insights Dashboard

An interactive data visualization project analyzing New York City taxi trip records to optimize operational revenue and identify rider behavioral trends.

## 🔗 Live Interactive Dashboard
👉 **[Click Here to View the Live Looker Studio Dashboard](https://datastudio.google.com/s/k_LdPNdlDNI)**

## 📊 Visualizations Practiced & Built
1. **Time Series Line Chart**: Tracks taxi fare revenue velocity over different calendar days.
2. **Clustered Column Bar Chart**: Compares average fare pricing structures grouped by specific passenger counts.
3. **Donut Chart**: Displays proportional trip volume segments across various taxi rate codes.
4. **Pie Chart**: Breaks down the total gross revenue share split by user transaction payment types.
5. **Scatter Plot Visual**: Correlates independent riding metrics to isolate cost outliers.
6. **Key KPI Scorecard Block**: Computes real-time average improvement surcharge metrics.
7. **Layered Volume Area Chart**: Maps volume distribution density across continuous timeline layers.

## 🛠️ Tech Stack & Tools
* **BI Platform:** Google Looker Studio
* **Data Engineering:** Microsoft Excel / CSV Parsing
* **Dataset Used:** NYC Taxi & Limousine Commission Yellow Trip Subset
